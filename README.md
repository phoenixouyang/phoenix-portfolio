# Phoenix Ouyang — Portfolio

My personal portfolio website built with HTML and modern CSS. It showcases my projects, freelance work, experience, and a little about who I am outside of code.

🔗 **Live Site:** [phoenix-portfolio-tau.vercel.app](https://phoenix-portfolio-tau.vercel.app/)

## About Me

I'm a Software Developer with a background in financial analysis — previously at Ubisoft, Lindt & Sprüngli, Unilever, and Brookfield Asset Management. After five years analyzing products, I wanted to be the one building them, so I completed a Computer Programming Diploma at Seneca Polytechnic and made the career switch. This portfolio highlights what I've built since.

## How It's Built

- **HTML5** — accessible landmark structure (`nav`, `main`, `section`, `footer`)
- **CSS** — design tokens via custom properties, CSS Grid with `auto-fit`/`minmax()` for self-reflowing card layouts, and fluid typography with `clamp()` so text scales smoothly across every viewport
- **Inter** typeface via Google Fonts
