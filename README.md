# Phoenix Ouyang — Portfolio
A personal portfolio website built with vanilla HTML, CSS, and JavaScript. Showcases my projects, work experience, and a little about who I am outside of code.

🔗 **Live Site:** [phoenixouyang.github.io](https://phoenix-portfolio-tau.vercel.app/)

## About
I'm a Software Developer with a background in financial analysis — previously at Ubisoft, Lindt & Sprüngli, Unilever, and Brookfield Asset Management. This portfolio highlights the projects I've built since transitioning into software development, as well as freelance work I've contributed to.

## Tech Stack
- **HTML5** — semantic markup and structure
- **CSS3** — custom styling, responsive layout, collage section
- **Vanilla JavaScript** — Intersection Observer API for scroll-based fade-in animations
- **Google Fonts** — Inter typeface
